How to use ?
Trick uses the following syntax for basic programming functionality along with some additional features:
  • Variables: use keyword var to declare variables. To reassign variables, omit the use of the keyword var
  • Operations: works the same as Java
  • Datatypes: There are 5 datatypes while are handled implicitly:
    • Boolean: true and false
    • Numbers: Handled as a double
    • String: Same as Java along with the added functionality of concatenation using +
    • Char: Same as Java
    • Nil: nil is the keywork to denote null type
  • Conditional: Use can use if() {}else if(){} else{} similarly to Java along with teritiary operation (condition) ? true : false
  • Loops: Looping is the same as Java with for(), while() and do{} while()
  • Print: For printing, use the print() command
